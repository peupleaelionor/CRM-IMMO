# EstateFlow — CRM Immobilier IA

CRM intelligent pour agents immobiliers, propulsé par l'IA. Matching acheteurs/biens automatique, génération de descriptions, pipeline visuel et automatisation.

## 🚀 Démarrage

```bash
npm install
npm run dev
```

## 📦 Build & Déploiement

```bash
npm run build   # → dist/
```

Déploiement automatique sur Netlify via GitHub Actions.

## 🛠 Stack

- React 19 + TypeScript
- Vite 8
- TailwindCSS 3
- React Router 7
- lucide-react (icônes)

## ✨ Fonctionnalités

- **Tableau de bord** — Vue d'ensemble avec suggestions IA
- **Contacts** — Gestion acheteurs/vendeurs avec scoring
- **Biens** — Portefeuille avec génération de descriptions IA
- **Pipeline** — Kanban drag & drop des deals
- **Matching IA** — Correspondance automatique acheteurs ↔ biens
